# Privacy Policy for Yuka Discord Bot

**Effective Date:** [Today’s Date]

Your privacy is important to us. This Privacy Policy explains how the Yuka Discord Bot ("the Bot") handles your data.

## 1. Data Collection
The Bot **does not collect, store, or share** any personal user data. All data is processed temporarily and in memory only to execute commands such as `!ping`, `!play`, etc.

## 2. Message Content
The Bot reads message content only when it starts with a recognized prefix (e.g., `!`). It does not store message history or user conversations.

## 3. Server Data
The Bot may temporarily access server information (such as server name, user count, or roles) as needed to perform its functions. This data is not stored or shared.

## 4. Third Parties
The Bot does **not share any data** with third parties.

## 5. User Rights
If you have questions or requests related to privacy, please contact the developer.

## 6. Changes
This Privacy Policy may be updated. You will be notified through the official support server if any significant changes occur.

## 7. Contact
For any concerns or questions, contact the developer via Discord: `@YourDiscordTag`

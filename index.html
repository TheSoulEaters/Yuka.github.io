Privacy Policy for Yuka Discord Bot

Effective Date: June 5, 2025

1. Data Collection

The Yuka Bot does not collect, store, or share any personal user data. All data is processed temporarily and in memory only to execute commands such as !ping or !play.

2. Message Content

The bot reads message content only when it begins with a valid prefix (e.g., !). No message history or private conversations are stored.

3. Server Data

Server data such as server name or user count may be accessed temporarily to support bot functions. None of this information is stored or transmitted outside Discord.

4. Third Parties

The bot does not share any data with third-party services.

5. User Rights

If you have any questions regarding privacy, please contact the developer directly via Discord.

6. Changes

This privacy policy may be updated in the future. Any significant changes will be communicated through the bot's support server.
